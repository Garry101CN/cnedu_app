# Android App - CN5

## 技术选型与架构设计
 - 编程语言：Kotlin，Java
 - MVVM框架：Jetpack
    - LiveData, Lifecycle, Databinding
    - WorkManager
    - Navigation
    - Paging
    - ViewPager 3
 - 网络框架：OkHttp, Retrofit
 - 图片框架：Glide
 - Json解析框架：Gson
 - 网络浏览器框架：WebView
 - 视频播放框架：ExoPlayer
 - 第三方辅助工具：utils, refresh, x5
